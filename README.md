# Worms VR Stage

This is a basic static A-Frame/WebXR scene for Meta Quest Browser.

The VR setup is adapted from `templates/simple-vr-scene` in
`https://github.com/2ndsebastiantablet-hash/feeble` at commit
`28a426aa6ade789320e2202cfa8d2fe61b46b539`.

## Run

Serve the folder from any static HTTPS host, open it in Meta Quest Browser,
and press the A-Frame `Enter VR` button.

No npm, framework, bundler, or build step is required.
